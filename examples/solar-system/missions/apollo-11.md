# Apollo 11

Apollo 11 was the #historic #crewed #nasa #mission that first landed humans on the Moon on July 20, 1969 — part of the #apollo-program. Commander Neil Armstrong and lunar module pilot Buzz Aldrin landed the Apollo Lunar Module *Eagle* while command module pilot Michael Collins orbited above in *Columbia*.

- **Buzz Aldrin** — Lunar Module Pilot
- **Michael Collins** — Command Module Pilot
